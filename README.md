# 空 (Sora) - Sky detox

「ただ流し、ただ消し、ただ見つめるだけ。」

画面を漂う色の変化と波紋を楽しむ、ミニマルなWebアート・アプリケーションです。日常の喧騒を離れ、静かな時間を過ごすためのツールとして作成しました。



## ✨ 特徴

- **流す (Flow):** 空の色を次の色へと滑らかに変化させます。
- **消す (Erase):** ランダムな空の色へ瞬時にリセットします。
- **波紋 (Ripple):** クリックした場所に広がる静かな波紋エフェクト。
- **レスポンシブ:** どんな画面サイズでも美しく表示されます。

## 🚀 使い方

1. `index.html` をブラウザで開きます。
2. 画面下部のボタン、または画面上のクリックで操作します。
3. 変化する空の色を眺めて、リラックスしてください。

## 🛠 技術スタック

- **HTML5 / CSS3**
  - CSS Transition による滑らかな色変化
  - Backdrop-filter を使用したガラス質（Glassmorphism）なUI
- **Vanilla JavaScript**
  - DOM操作とイベントリスナーによるシンプルな制御

## 🎨 コンセプト

デジタルな世界において、あえて「何もしない時間」を作ることを目的としています。特定の目的を持たず、ただ色の移り変わりを眺める体験を提供します。

## 💡 こだわりポイント

- **色の選定:** 10種類のグラデーションは、実際の空の移り変わり（夜明け、正午、黄昏時など）をイメージして厳選しました。
- **2秒間の余白:** 色が切り替わる際に `2s` の時間をかけることで、急激な変化ではなく、自然界のようなゆっくりとした移ろいを表現しています。
- **触感:** ボタンを押したときの波紋（Ripple）エフェクトにより、デジタルな画面に水面のような「手応え」を与えています。

## 🧘 私の想い

忙しい毎日の中で、ふと空を見上げる時間を忘れてしまうことがあります。
このアプリは、そんな時にブラウザのタブ一つで「心の呼吸」ができる場所として作りました。
ただ流れる色を眺める、その「無意味な時間」の豊かさを感じていただければ幸いです。

## 🛠 実装の工夫（Technical Notes）

- **メモリへの配慮:** 波紋エフェクトで生成された `div` 要素は、アニメーション終了後に `setTimeout` で確実に DOM から削除し、ブラウザの負荷を抑えています。
- **数学的ランダム:** `Math.random()` を使用し、「消す（リセット）」を押すたびに異なる表情の空に出会えるように設計しました。
- **デザイン:** ボタンには `backdrop-filter: blur()` を採用し、背景の色を透過させつつ文字の視認性を確保する「グラスモーフィズム」を取り入れています。
